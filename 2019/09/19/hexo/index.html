步骤：

安装

$ npm install hexo-cli -g
安装后检查是否安装成功：
$ hexo -v
成功后进行初始化：
$ hexo init myBlog
安装组件:

node_modules: 依赖包
public：存放生成的页面
scaffolds：生成文章的一些模板
source：用来存放你的文章
themes：主题
** _config.yml: 博客的配置文件**

$ hexo g
$ hexo s

deploy:
type: git
repo: https://github.com/YourgithubName/YourgithubName.github.io.git
branch: master

$ npm install hexo-deployer-git –save

$ hexo clean
$ hexo generate
$ hexo deploy

$ hexo clean
$ hexo d -g
